# Vladis Mobil Grooming

Vladis Mobile Grooming Website<br/>
[Visit Website](https://vladis.netlify.app/)<br/>
Vladis Mobile Grooming provides Mobile Dog grooming<br/>

## Tech Stack

**Language**:
TypeScript

**Framework**:
Gatsby, React, React Formik & Yup

**Styling**:
Scss, TailwindCss

**Data storage**:
Strapi Headless Cms
